The Cosmos Programming Language

Creating the Cosmos

Cosmos is a language born out of frustration with other languages. After falling into the black holes of other languages for years, Cosmos was finally born.

Terminology

The Cosmos is made of Matter:

  • Solid: Compiled code, may be used as a library or as the entry point of execution. The file extenson is .csolid
  • Liquid: Source code in raw form. The file extension is .cliquid
  • Gas: Declarations-only file, similar to C++ headers. The file extension is .cgas
  • Fluid: Fluids are liquids and gasses, which which make up a program.
  • Solidification: Compilation of liquids into solids
  • Sublimation: Creating a gas from a solid.
  • Liquefy: To execute a program.
  • Condensation: Implementing a gas in a liquid.
  • Evaporation: Creating a gas from a liquid.
