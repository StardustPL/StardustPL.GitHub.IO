The PolyStatic Programming Language

What makes PolyStatic different?

PolyStatic is targeted at static typesafety and runtime modularity. The goal is to have a type system similar to those of C++ and Java, but runtime compatibility like that of Python. The intended use is programming game logic.

How do I learn to write PolyStatic code?

The documentation isn't read yet because the language is still very new. There are examples, though.

How do I compile/run it?

You have to use the interpreter for now, more options will be available later (such as compiling and running live from existing expplications).

What's the Hello World program?

For there to be a Hello World program, there has to be a way to output information somwhere. PolyStatic is meant to be used by other code (e.g. a game), so it depends on the caling code to provide e.g. a debugging callback function.
